# TECNICALTESTING APP

## Installation
To install follow next steps:

### 1) Create a virtual enviroment.
-**LINUX**

``` source .venv/bin/activate ```

-**WINDOWS**

``` python -m venv .venv ```

### 2) Install dependencies.

``` pip install -r requirements.txt ```

## RUN APP

### 1) Activate virtual enviroment.

### 2) Run app.

``` python __init__.py ```
